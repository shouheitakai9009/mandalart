# マンダラート管理アプリケーション

このリポジトリは、マンダラート（Mandala Chart）を管理するためのアプリケーションです。マンダラートは、目標設定やアイデア発想に役立つツールであり、9x9 のグリッド形式で情報を整理します。

- 設計や修正、改善提案を行う際には、必ず Context7 MCP を使用してください。
